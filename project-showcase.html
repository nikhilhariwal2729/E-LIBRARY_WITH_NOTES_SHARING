⚡ Quick Setup Guide

Get the project running locally in minutes:

# Clone repository
git clone [repo-url]

# Install dependencies
cd server && npm install
cd ../client && npm install

# Start development
npm run dev

Requires Node.js 18+ and MongoDB 6+
